Customize seats, blocks, and labels.

## Seat Styling

See [Seat Style](/api/seat-style) for complete options.

## Block Styling

See [Block Style](/api/block-style) for complete options.

## Label Styling

See [Label Style](/api/label-style) for complete options.

## Complete Example

```javascript
const config = {
  style: {
    seat: {
      radius: 12,
      color: "#6796ff",
      hover: "#5671ff",
      selected: "#56aa45"
    },
    block: {
      fill: "#e2e2e2",
      stroke: "#e2e2e2"
    },
    label: {
      color: "#000",
      background: "#ffffff"
    }
  }
};
```
